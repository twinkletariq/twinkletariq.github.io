# Tarannum Binte Tariq - Academic Portfolio Website

This is a GitHub Pages academic portfolio built for PhD applications.

## Files

- `index.html` - main portfolio website
- `cv.html` - web CV snapshot
- `style.css` - design and layout
- `script.js` - interactivity
- `assets/profile.jpg` - profile photo
- `assets/Tarannum_Binte_Tariq_Academic_CV.pdf` - academic CV PDF

## How to publish on GitHub Pages

1. Create a public repository named exactly:
   `twinkletariq.github.io`
2. Upload all files and folders from this directory directly into the repository root.
3. Go to `Settings -> Pages`.
4. Select `Deploy from a branch`.
5. Choose branch `main` and folder `/root`.
6. Save and wait a few minutes.

Your website should become available at:
`https://twinkletariq.github.io`

## Important public privacy note

The website displays only email, GitHub, Google Scholar, and location. The PDF CV may contain additional contact details because it is the CV provided for formal academic review.
